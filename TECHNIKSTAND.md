# Technikstand und Freigabe

Stand: 18. Juli 2026. Das Register enthält nur Quellenreferenzen und technische Prüfdaten, keine kopierten Normtexte. Eine Quellenabweichung wird erst durch eine zuständige Person an der Originalquelle geprüft; erst danach werden Skill oder Vorlagen geändert. Eine DIN-Referenz ist keine pauschale gesetzliche Pflicht oder automatische Freigabe.
