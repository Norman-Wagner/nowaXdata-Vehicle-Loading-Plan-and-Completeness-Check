# Changelog

## 0.1.0 – 2026-07-18

- Initial public skill with methodology, schemas, safety/privacy controls, templates, portable prompts, tests, and automated validation.
- Added compatible Open Agent Skills identifier and Claude Code plugin metadata.
- Added the original five-dimension readiness model: completeness, condition, validity, location, and requirement evidence.
- Added dynamic storage positions for drawers, sliding or lifting mechanisms, removable modules, travel state, retention checks, and access sequences.
- Added a Germany-specific source-discovery guide covering BALM (formerly BAG), official law portals, DGUV, accident-insurance institutions, BASt, technical standards, and manufacturer documentation without treating any source family as automatically applicable.
- Added a German funeral-vehicle jurisdiction router covering DIN 75081, all sixteen state research paths, transport scenarios, and verified examples from Saxony and Bavaria without reproducing copyrighted standard text.
- Added scoped research guidance for public procurement specifications and official police, fire, rescue, civil-protection, recovery, and municipal vehicle concepts; examples cannot be generalized into legal duties or copied load plans.
- Added a distinct `CERT` evidence class and source routing for the Markenzeichen der Bestatter, the BDB certification program, DIN EN 15017, DAkkS program status, audit evidence, and vehicle-related certification checks without exposing organization or case data.
