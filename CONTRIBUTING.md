# Contributing

Contributions are welcome through issues and pull requests. Use fictional examples. Never submit people, customer/case data, health data, access codes, confidential numbers, or identifiable photos.

Claims presented as legal, technical, accident-insurance, or manufacturer requirements need an exact current source. Otherwise label them internal, recommended, or unverified.

Run `python scripts/build_portable_prompts.py --check`, `python scripts/validate_repo.py`, and `python -m unittest discover -s tests -v`.
